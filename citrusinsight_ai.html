CitrusInsight AI - Citrus Disease Detection System

About the Project

CitrusInsight AI is an AI-powered citrus leaf disease detection system designed to assist citrus farmers by detecting diseases from leaf images. Our project utilizes machine learning models to classify leaves into 10 different disease categories.

Key Features

  • Citrus leaf disease classification (10 classes)
  • Image upload and disease prediction interface
  • Performance metrics: Precision, Recall, F1-Score, Confusion Matrix
  • Web-based, accessible via browser
